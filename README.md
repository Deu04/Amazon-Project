# Amazon Sales Dashboard 📊

## Project Overview

The Amazon Sales Dashboard is an interactive Business Intelligence project developed using Power BI to analyze and visualize Amazon sales performance across different regions, product categories, and customer segments.

The dashboard transforms raw sales data into meaningful insights that help identify sales trends, top-performing products, profitable regions, and business growth opportunities.

## Key Features

* 📈 Sales Performance Analysis
* 🌍 Region-wise Sales Insights
* 📦 Category & Sub-Category Analysis
* 💰 Profit and Revenue Tracking
* 📅 Time-Based Sales Trends
* 🎯 KPI Monitoring Dashboard
* 🔍 Interactive Filters and Drill-Down Analysis

## Dashboard Insights

* Total Sales Overview
* Total Profit Analysis
* Top Performing Categories
* Best Selling Products
* Regional Sales Distribution
* Profitability by Category
* Monthly and Yearly Sales Trends

## Tools & Technologies Used

* Power BI
* Microsoft Excel
* Data Cleaning & Transformation
* Data Visualization
* DAX (Data Analysis Expressions)
* Business Intelligence Techniques

## Business Value

This dashboard helps stakeholders:

* Monitor sales performance in real time
* Identify high-profit products and categories
* Analyze regional sales patterns
* Make data-driven business decisions
* Track overall business growth
